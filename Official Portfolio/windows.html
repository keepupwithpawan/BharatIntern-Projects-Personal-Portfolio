Windows 11 Clone

Personal Project | Front-End Web Development | UI/UX Design (In Proress)

It is a personal project inspired by a YouTube video I saw. Windows 11 has a really minimal and aesthetic looking UI, and I consider minimal and aesthetic as my niche.

This project is an attempt to bring the home screen of Windows 11 to life. All the taskbar links are clickable and serve a function. The future plan is to expand the project and make a completely web based working OS experience that would be seamless to work with.

Technologies used: HTML,CSS,JavaScript
