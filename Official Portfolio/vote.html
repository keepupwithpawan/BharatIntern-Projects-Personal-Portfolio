Online Voting System

Academic Project | Front-End Web Development | UI/UX Design (In Progress)

Its is a mini project submission for our academic project of second semester made for keeping in mind the student council elections of our college.

I worked on this project with a team of 3 other classmates where my role was to design the UI and front end development. This project was made so that students can vote for thier desired candidate without having to attend the college or compromise the leaks of the votes by submitting a google form. This website keeps security of choices in mind and dis made such that only the admin can see how many votes each party/candidate has.

Technologies used: HTML,CSS,PHP
